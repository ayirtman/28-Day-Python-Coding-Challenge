# Day 1 Python Challenge: Hello World, Variable Types, IO Operations

This is the first day of the Python Challenge, focusing on basic Python programming skills including:
- Printing a hello world statement
- Working with variable types
- Performing basic input and output operations

## Exercise

The goal for today is to develop a Python program that:
- Asks for a user's name
- Asks for a user's age
- Asks for a user's favorite color

After obtaining these inputs, your program should output a friendly greeting that includes the provided details.

## Bonus

For an extra challenge, modify your program to also ask for the user's country of residence and their favorite food, and include this information in the output.
